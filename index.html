Git

  • git config --global user.name: helps you configure your git settings for the name associated with each commit.
  • git config --global user.email: helps you configure your git settings for the email associated with each commit.
  • git status: shows the modified files in the working directory. Also, staged for your next commit.
  • git checkout: will switch to another branch and check it out into your working directory.
  • git commit -m "enter unique message": will commit your staged content as a new commit snapshot.
