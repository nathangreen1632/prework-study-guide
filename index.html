HTML

  • The head element contains information about the webpage, or "metadata".
  • The body element represents the visible content shown to the user.
  • The declaration type is not case sensitive.
  • HTML attibutes are always defined in the beginning element, never the closing element.
  • TAG is applying a certain format to a markup language.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git config --global user.name: helps you configure your git settings for the name associated with each commit.
  • git config --global user.email: helps you configure your git settings for the email associated with each commit.
  • git status: shows the modified files in the working directory. Also, staged for your next commit.
  • git checkout: will switch to another branch and check it out into your working directory.
  • git commit -m "enter unique message": will commit your staged content as a new commit snapshot.

JavaScript

  • An array is a single variable that is used to hold a group of data.
  • A unique number in an array is called an index.
  • In JavaScript and many other programming languages, arrays are zero-indexed and sequential. This means that the first item in any array has an index of 0, not 1. The indices, or location identifiers, will always be whole numbers that increase by 1 each time.
  • A "for" loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
